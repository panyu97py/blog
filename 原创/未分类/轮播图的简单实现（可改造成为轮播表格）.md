# 轮播图的简单实现（可改造成为轮播表格）

### 水平方向的轮播

```
<template>
    <div >
      <div >
        <div v-for="item in testList"
            :key="item.color"
            :style="`background:${item.color};`"
            >
            {{item.text}}
        </div>
      </div>
    </div>
</template>
<script>
export default {
  data () {
    return {
        testList: [
          {text: '1', color: 'red'}, 
          {text: '2', color: 'yellow'}, 
          {text: '3', color: 'pink'}, 
          {text: '4', color: 'blue'}
          ]
        }
  },
  mounted () {
    setInterval(() => {
      const tempItem = this.testList.splice(0, 1)
      this.testList.push(...tempItem)
    }, 1000)
  }
}
</script>
<style lang="scss" scoped>
  .view {
    //需要显示的高度
    //需要显示的宽度
    overflow: hidden;
    &_body {
      margin-left: -100px;//item的宽度
      display: flex;
    }
  }

  .item {
    
    
    flex-shrink: 0;//防止挤压变形
    transition: all 1s;
    text-align:center;
    line-
    &:first-child {
      width: 0;
    }
  }
</style>
```
